## TXT
1. Создать внешний репозиторий c названием TXT `create a new repository xml`
 2. Клонировать репозиторий TXT на локальный компьютер git clone `https://github.com/ESKovalenko/TXT.git`
 3. Внутри локального TXT создать файл “new.txt” `vim new.txt`
 4. Добавить файл под гит `git add `new.txt`
 5. Закоммитить файл `git commit -m "new.txt"`
 6. Отправить файл на внешний GitHub репозиторий `git push`
 7. Отредактировать содержание файла “new.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT `vim new.txt`
 
Full name - Коваленко Екатерина Сергеевна

Age - 26

Number of pets - 2

Salary - 1000`

 8. Отправить изменения на внешний репозиторий `git add new.txt` `git commit -m "new.txt"` `git push`
 9. Создать файл preferences.txt `vim preferences.txt`
 10. В файл preferences.txt” добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT `vim preferences.txt`
 
Favorite_movie - A Dog's Purpose

Favorite_series - The Big Bang Theory

Favorite_food - Sushi

Favorite_season - Summer

The Country I Want To Visit - Australia

 11. Создать файл sklls.txt добавить информацию о скиллах которые будут изучены на курсе в формате TXT `vim skills.txt`

HTTP, API, Postman, Charles, Android Studio, Jmeter

 12. Сделать коммит в одну строку `git add .| git commit -m "preference and skils"`
 13. Отправить сразу 2 файла на внешний репозиторий `git push`
 14. На веб интерфейсе создать файл bug_report.txt `create new file`
 15. Сделать Commit changes (сохранить) изменения на веб интерфейсе `comite change`
 16. На веб интерфейсе модифицировать файл bug_report.txt, добавить баг репорт в формате TXT 
Bug ID

Bug Summary

Bug Details

Bug Priority

Bug Description

Attachments to the bug report

 17. Сделать Commit changes (сохранить) изменения на веб интерфейсе `commit change`
 18. Синхронизировать внешний и локальный репозиторий TXT `git pull`
